# Sales Data Analysis Project
## Project Description
This project demonstrates a practical **Sales Data Analysis** using Microsoft Excel.  
It includes an **interactive dashboard** and several **Pivot Tables** to explore sales performance across regions, categories, and customer types.  

**Tools & Technologies:** Microsoft Excel 365, Pivot Tables, Charts, Dashboard  

## Dataset Overview
The dataset contains multiple dimensions and metrics:  

| Feature | Description |
|---------|-------------|
| Transactions | Date, Month, Product, Category |
| Sales Team | Sales representatives, Region managers |
| Regions | East, West, North, South |
| Financial Metrics | Quantity Sold, Unit Price, Total Sale, Cost Price, Profit |
| Customer Segmentation | New, Regular, VIP |
| Payment Methods | Cash, Credit Card, Online Transfer |
| Order Status | Completed, Pending, Canceled |

## Analysis Methods
- **Pivot Tables**: Summarized sales by region, category, representative, and customer type.  
- **Charts**: Bar charts, line graphs, and pie charts to visualize trends and performance.  
- **Interactive Dashboard**: Filters for customer type, payment method, and region to explore data dynamically.  

## Key Insights
- **Top Revenue Category**: Sports generated the highest revenue.  
- **Regional Performance**: South region contributed the largest share of total sales.  
- **Order Status**: Most orders were completed successfully.  
- **Seasonal Trends**: Monthly trends revealed peaks during Q1 and Q2.
